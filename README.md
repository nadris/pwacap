# Diseño Frontend del CAP

Diseño de PW (Centro de Autoempleo y Produccion). usando HTML y tecnicas CSS3, flexbox, responsive,BEM.

## ---> install

```
npm install --save
```
## ---> run
```
node server.js
```
## ---> open

```
http://localhost:5454/home.html
http://localhost:5454/notificacion.html
```
